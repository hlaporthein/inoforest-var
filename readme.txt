== Description ==
 
The VarDumper component provides mechanisms for extracting the state out of any PHP variables. Built on top, it provides a better dump() function that you can use instead of var_dump.

You would need to use dd() or dump() methods in your plugins or theme development. No other feature are not included just synfony var dump.
 
== Installation ==
 
= Minimum Requirements = 
* WordPress 5.0 or greater
* PHP version 7.2 or greater
 
=Installation=
 
1. Upload `inoforest-var` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You could print now synfony `dd() or dump()` in your templates or plugin development
